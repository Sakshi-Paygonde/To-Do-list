# 📝 To-Do List Web App

A simple and beautiful To-Do List app that lets you:
- Add tasks with deadlines
- Mark them as complete
- Delete them  
All data is saved in your browser using localStorage.

🌐 **Live Demo**:  
👉 [Open the App](https://sakshi-paygonde.github.io/To-Do-list/)

📦 **GitHub Repository**:  
👉 [View on GitHub](https://github.com/Sakshi-Paygonde/To-Do-list)

---

## ✨ Features

- ✅ Add tasks with deadlines  
- 🎯 Mark as completed  
- 🗑️ Delete tasks  
- 💾 Auto-save with `localStorage`  
- 🎨 Soft pastel theme & responsive design

---

## 💻 Tech Stack

- HTML  
- CSS  
- JavaScript
